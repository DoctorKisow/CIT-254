Part 1 - Scanning

Steganography is a way to hide information within another message or physical object. The steghide program is a steganography application that can hide data in various kinds of image files and audio files. The color sample frequencies are not changed, thus making the embedding resistant against first-order statistical tests.

  1. Update the Kali Linux operating system to ensure it runs the latest packages.
    ⬤ ⬤ ⬤ CODE

      # Update Kali Linux
       $ sudo apt update
      $ sudo apt -y upgrade

  1. Install the steghide application.
    ⬤ ⬤ ⬤ CODE

      # Install Steghide
       $ sudo apt -y install steghide

REFERENCES

  • https://www.kali.org/tools/steghide/

Part 2 - Exploit

Using your book and the Internet as a reference; read the embedded contents of the "drkisow-message.jpg" file, then create your own secret message by hiding it in the "ccaclogo.jpg" file.

  1. Using the Thunar File Manager open the "/labs/LAB9" folder and copy the "drkisow-message.jpg" file to your on your Kali Linux VM desktop.
  1. Open your command prompt and extract (decode) the contents of the "drkisow-message.jpg" file to the "drkisow-message.txt" file using CIT254 as the passphrase to extract the hidden contents.
    ⬤ ⬤ ⬤ CODE

      # Change Directory
       $ cd /home/labuser/Desktop

      # Extract Hidden Message
       $ steghide extract -sf drkisow-message.jpg -xf drkisow-message.txt
      Enter passphrase:
       wrote extracted data to "drkisow-message.txt".

Part 3 - Post-Exploit

Using your book and the Internet as a reference; encode a secret message in the "ccaclogo.jpg" file.

  1. Using the Thunar File Manager open the "/labs/LAB7" folder and copy the "ccaclogo.jpg" file to your on your Kali Linux VM desktop.
  1. Using vi, nano, or vim> create a new text file called "secret.txt" on your Kali Linux VM desktop; write your own favorite quote similar to mine. Ensure you save it, then close the file.
  1. From your command prompt and embed the contents of the "secret.txt" file to the "ccaclogo.jpg" file encoding it with the CIT254 as the passphrase.
    ⬤ ⬤ ⬤ CODE

      # Change Directory
       $ cd Desktop

      # Encoding a Hidden Message
       $ steghide encode -ef secret.txt -cf ccaclogo.jpg
      Enter passphrase:
       Re-Enter passphrase:
       embedding "secret.txt" in "ccaclogo.jpg" ... done

     

    NOTE
      ●

    Capitalization is important; type the passphrase as it is shown.
  1. Rename the "ccaclogo.jpg" file to "<lastname>-message.jpg".
    ⬤ ⬤ ⬤ CODE

      # Rename File
       $ mv ccaclogo.jpg <lastname>-message.jpg
