Lab Report Format

Each lab will have a set of deliverable items that you must format in place into a lab report. For each deliverable item, create a number in your report that corresponds with the deliverable question. For example in the following deliverable item, the lab report would have the following sections:

 

DELIVERABLE

  1.1
  A.

What command did you type to create a new user on the system?
  i.

Include a screenshot of the command that you typed.

John Q. Student
Lab 1

April 20, 2024


1.1 A   : useradd someuser
1.1 A(i): <Inserted screenshot of the executed command>
