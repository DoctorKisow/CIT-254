Part 1 - Installing Steghide

Steganography is a way to hide information within another message or physical object. The steghide> program is a steganography application that can hide data in various kinds of image files and audio files. The color sample frequencies are not changed, thus making the embedding resistant against first-order statistical tests.

  1. Install Steghide:
    1. Update Kali Linux: Update the Kali Linux operating system to ensure it runs the latest packages.
    2. Install Steghide: To install steghide, follow the procedure.
      ⬤ ⬤ ⬤ CODE

        # Update Kali Linux
         $ sudo apt update
        $ sudo apt -y upgrade

      ⬤ ⬤ ⬤ CODE

        # Install Steghide
         $ sudo apt -y install steghide

REFERENCES

Please read this article before you try to install steghide.

  • https://www.kali.org/tools/steghide/

Part 2 - Writing and Hiding a Message

Using your book (p. 370-371) and the Internet as a reference, we will read the embedded contents of a file, then create a text file and hide it in the "ccaclogo.jpg" file attached to this lab.

  1. Reading the contents of the "drkisow-message.jpg" file.
    1. Download the "drkisow-message.jpg" file to your Kali Linux desktop.
    2. Open your command prompt and extract the contents of the "drkisow-message.jpg" file to the "drkisow-message.txt" file.
    3. The passphrase is CIT254.
    4. Screenshot the contents of this text file; you will need it later.
      ⬤ ⬤ ⬤ CODE

        # Change Directory
         $ cd Desktop

        # Extract Hidden Message
         $ steghide extract -sf drkisow-message.jpg -xf drkisow-message.txt
        Enter passphrase:
         wrote extracted data to "drkisow-message.txt".

  2. Create your own hidden message using the "ccaclogo.jpg" file.
    1. Download the "ccaclogo.jpg" file to your Kali Linux desktop.
    2. Create a new file, "secret.txt," on your Kali Linux desktop.
    3. Open the "secret.txt" file and write your own favorite quote similar to mine. Ensure you save it, then close the file.
    4. Open your command prompt and embed the contents of the "secret.txt" file to the "ccaclogo.jpg" file.
    5. Use the passphrase CIT254 to encode it. Capitalization is important; type the passphrase as it is shown.
    6. Rename the "ccaclogo.jpg" file to "<lastname>-message.jpg".
      ⬤ ⬤ ⬤ CODE

        # Change Directory
         $ cd Desktop

        # Encoding a Hidden Message
         $ steghide encode -ef secret.txt -cf ccaclogo.jpg
        Enter passphrase:
         Re-Enter passphrase:
         embedding "secret.txt" in "ccaclogo.jpg" ... done

         # Rename File
         $ mv ccaclogo.jpg <lastname>-message.jpg
